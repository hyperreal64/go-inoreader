Inoreader command structure:
- [ ] `inoreader` Main command. If no args supplied, display usage info
   - [X] `login` - Initiate OAuth, save info to config @done(21-02-16 20:01)
   - [ ] `help` - Display help info
   - [X] `add <streamID>` - Add subscription @done(21-02-16 14:44)
   - [X] `sub` - Query subscriptions; requires exactly one of below args @done(21-02-13 15:50)
      - [X] `all` - List subscriptions @done(21-02-13 15:50)
      - [X] `unread` - List only unread subs with counts @done(21-02-13 15:50)
      - [X] `edit` - Edit a subscription @done(21-02-16 16:45)
          - [X] `-a <edit|subscribe|unsubscribe>` - Edit, subscribe, or unsubscribe from sub. REQUIRED @done(21-02-16 16:45)
          - [X] `-s <streamID>` - Stream ID of subscription. REQUIRED @done(21-02-16 16:45)
          - [X] `-t <title>` - Change the subscription title. Omit to keep title unchanged. @done(21-02-16 16:45)
          - [X] `-f <folder>` - Add subscription to folder. Use folder name like `user/-/label/Tech` @done(21-02-16 16:45)
          - [X] `-r <folder>` - Remove subscription from folder. Use folder name like `user/-/label/Tech` @done(21-02-16 16:45)
  - [X] `tags` - Query tags and folders; requires exactly one of below args @done(21-02-13 19:45)
      - [X] `all` - List all tags and folders; optional below args @done(21-02-13 19:45)
          - [X] `-t|--type <tags|folders>` - Specify and list only tags XOR folders (string) @done(21-02-13 19:45)
      - [X] `unread` - List only folders containing unread items with counts @done(21-02-13 19:45)
          - [X] `-t|--type <tags|folders>` - Specify and list only tags XOR folders with unread counts @done(21-02-15 14:37)
      - [X] `edit` - Edit a tag on an item @done(21-02-16 16:02)
          - [X] `-a <tag>` - Tag to add @done(21-02-16 16:02)
          - [X] `-r <tag>` - Tag to remove @done(21-02-16 16:02)
          - [X] `-i <itemID>` - Item ID. @done(21-02-16 16:02)
      - [X] `mv` - Rename a tag @done(21-02-16 16:45)
          - [X] `-s <source tag>` - Source tag's name @done(21-02-16 16:45)
          - [X] `-d <dest tag>` - Destination tag's name @done(21-02-16 16:45)
      - [X] `rm <tag name>` - Delete a tag. Use full tag name like `user/-/label/Tech` @done(21-02-16 16:46)
  - [X] `stream` - Query stream contents; requires at least one of below args @done(21-02-15 15:37)
      - [X] `-n <int>` - Specify numbers of items returned @done(21-02-15 15:37)
      - [X] `-r <newest|oldest>` - Specify order of items displayed @done(21-02-15 15:37)
      - [X] `-xt <streamID>` - Specify streamID of label to exclude @done(21-02-15 15:37)
      - [X] `-it <streamID>` - Specify streamID of label to include @done(21-02-15 15:37)
      - [X] `-s <streamID>` - Specify streamID of target @done(21-02-15 15:37)
      - [X] `-u` - Display with URL @done(21-02-15 19:59)
  - [X] `read <streamID>` - Mark a stream as read @done(21-02-15 17:15)
  - [ ] `output <streamID|itemID>` - Fetch and save content to output format; requires at least one of below args
      - [ ] `-m` - Export to Markdown
      - [ ] `-c` - Export to CSV
      - [ ] `-j` - Export to JSON
